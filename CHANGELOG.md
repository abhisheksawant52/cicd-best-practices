# Changelog

All notable changes to this project are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and this project
adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-07-08

### Added

- `cicd-lint` CLI with text, JSON, and SARIF output.
- Rules: pinned-actions, least-privilege-permissions, job-timeouts,
  concurrency-control.
- Hardened reusable workflow templates under `templates/workflows/`.
- Documentation, CI pipeline, pre-commit hooks, and open-source project files.
